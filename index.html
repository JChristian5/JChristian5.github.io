About Me

Coding Journey

During the 2020 pandemic, I found myself with a lot of extra time. One thing that caught my interest was coding since I had worked on it on and off in the past. After talking to a friend, he suggested to start learning Python. While I was enjoying the book I worked through, I wanted to try my hand at front-end web development since I am more of a visual learner. After diving into the world of front-end development. I found a regained passion for web development since I used to use website builders when I was younger as a hobby. Ever since I switched to front-end web development, I knew that this was my passion.

Education

While I attended college, I did not pursue a technology degree. Before college, I used to work on easy websites using website builders, as well as doing small coding tasks. When I first started taking coding seriously in 2020, I started with Zed Shaw's book, Learn Python the Hard Way. After going through the book, I started to go through the freeCodeCamp curriculum and started to create small projects. I also found learning the basics of JavaScript a bit easier since I went through the Python book. I also eventually bought a Udemy course, Colt Steele's Web Developer Bootcamp to get more into DOM Manipulation.
